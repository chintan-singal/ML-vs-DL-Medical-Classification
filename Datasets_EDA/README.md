# Medical Image Analysis Project

## 👤 Member 1: Harsh (EDA & Data Preparation)

This repository contains the work related to Exploratory Data Analysis (EDA) and preprocessing for three medical image datasets.

---

## 📊 Datasets Used

1. Chest X-ray Pneumonia
2. Brain Tumor MRI
3. Skin Cancer (HAM10000)

---

1. 🫁 **Chest X-ray Pneumonia Dataset**  
   🔗 https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia  

2. 🧠 **Brain Tumor MRI Dataset**  
   🔗 https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset  

3. 🧬 **Skin Cancer (HAM10000) Dataset**  
   🔗 https://www.kaggle.com/datasets/kmader/skin-cancer-mnist-ham10000  

---

## 🔍 Work Done

- Exploratory Data Analysis (EDA)
- Class distribution analysis
- Image visualization
- Image preprocessing:
  - Resizing
  - Normalization
  - Label encoding
- Dataset understanding and comparison
- Preparation of data for:
  - Classical ML models (SVM, Logistic Regression, Random Forest)
  - Deep Learning models (CNN)
